A Homeland Security System which detects an intruder and sends an immediate image to the registered email(s) followed by a 10 seconds video.

Components Used:

Raspberry Pi 3B+
Raspberry Pi Camera Module V2
PIR Sensor (for motion detection)
A 5V 2.5A power supply for the Raspberry Pi
A sd card loaded with Raspbian Stretch OS
An external monitor or Laptop for accessing the CLI or GUI
HDMI cable
